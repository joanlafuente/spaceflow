Geometric Control Study

You will compare pairs of 3D objects generated from the same geometric input, using different levels of shape control.

Before you begin

How this study works

In each trial, a model generates a 3D asset from a text prompt and an input geometric shape. The input shape is divided into two control regions:

● Orange regions — strongly controlled. The generated object should stay close to their shape.
○ Grey/white regions — weakly controlled. The model may generate freely to match the prompt.

You will compare pairs of generated objects and answer three questions. Judge each question independently. A good result should match the prompt and preserve the intended structure in the orange regions.
